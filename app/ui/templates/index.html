PredixPy + Predix Cache

This is a quick demonstration for using the Predix Python SDK in combination with the Predix Cache service.

If you GET the endpoint /api/1.0/cache/$key for any valid url $key it will return the value stored for that key. If you want to replace or add data, you POST to the same endpoint.
